MediaTomb is an open source (GPL) UPnP MediaServer with a nice web user interface.

1. Install mencoder and sqlite3
sudo apt-get install mencoder libsqlite3-dev

2. Make
cd $COLO/apps/mediatomb
./mk

3. Start server and client.

The conf file of this server is COLO/apps/mediatomb/.mediatomb/config.xml.
